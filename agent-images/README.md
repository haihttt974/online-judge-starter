# Agent Images

Thu muc nay dung de luu anh chup man hinh, anh loi, anh trang thai UI hoac anh minh hoa chuc nang khi can agent xem lai hien trang de sua project chinh xac hon.

## Cach dung

- Dat anh vao thu muc nay truoc khi hoi agent ve mot trang thai UI/chuc nang.
- Neu anh lien quan den mot chuc nang cu the, nen dat ten theo mau:

```text
YYYY-MM-DD-ten-chuc-nang-mo-ta-ngan.png
```

Vi du:

```text
2026-05-01-judge-result-wa-filter.png
2026-05-01-upload-testcase-error.png
2026-05-01-mobile-layout.png
```

## Quy uoc

- Uu tien anh `.png` hoac `.jpg`.
- Moi anh nen the hien ro khu vuc can sua, thong bao loi, input dang dung va ket qua hien tai.
- Neu co nhieu anh cho cung mot loi, them so thu tu cuoi ten file, vi du `2026-05-01-upload-error-01.png`.
- Khi project thay doi UI/flow lon, cap nhat lai `Agent.md` neu quy uoc anh hoac vi tri thu muc thay doi.
